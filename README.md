# MOOMIN
Moment-Objective Minimum-Discrepancy (MOOMIN) Prior

![MOOMIN](https://github.com/user-attachments/assets/b2b65d4b-1723-4340-ae8a-5486aacc64e2)


This repository contains the R code used for the real data application in the manuscript:

> Rubio, F.J. (2026). An objective non-local prior for skew-symmetric models.

1. `routines_sn.R` : This code contains the routines required for the real data application and simulations.
2. `Application` : This folder contains the R code and R Markdown code and html to reproduce the real data application.
3. `Plots` : This folder contains the R codes to reproduce all the plots in the manuscript and appendix.
4. `routines` : This folder contains all routines used in the manuscript and appendix.

See also:
- [The effect of the shape (skewness) parameter in skew-symmetric models](https://rpubs.com/FJRubio/DTVMINSS)
- [The effect of the shape (skewness) parameter in skew-symmetric models, Part II](https://rpubs.com/FJRubio/DiscMinSS)
- [The effect of the shape (skewness) parameter in skew-symmetric models, Part III](https://rpubs.com/FJRubio/DivMinLC)
